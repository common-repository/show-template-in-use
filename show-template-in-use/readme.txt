== Description ==

(simply) show the template that is serving the current page - just activate to use! Please consider <a href="https://ko-fi.com/kriby">buying me a coffee <3 </a>

== Installation ==

Install the plugin, activate it, and a bar will render at the top of your pages that show which template is currently in use.  Please note this will only show for users who are logged in.
